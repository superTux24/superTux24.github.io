💡 Comment ça marche ?

Pour les utilisateurs : Naviguez simplement entre les deux sections du dictionnaire pour trouver vos traductions. Utilisez la barre de recherche pour trouver rapidement un mot spécifique.

Pour les contributeurs : Cliquez sur "✏️ Modifier le dictionnaire" pour accéder à l'interface d'administration. Vous pourrez alors :

  • Ajouter de nouveaux mots et traductions
  • Modifier les entrées existantes
  • Organiser par catégories
  • Exporter/importer des données

Aucune compétence technique requise ! L'interface est conçue pour être utilisée par tout le monde, même sans connaissances en informatique.
