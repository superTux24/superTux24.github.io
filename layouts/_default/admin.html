🔄 Actions rapides

📖 Mode d'emploi simple

🎯 Pour ajouter des mots :

  1. Cliquez sur l'onglet "➕ Ajouter un mot"
  2. Choisissez la direction (Français → Danois ou Danois → Français)
  3. Remplissez le mot et sa traduction
  4. Ajoutez une catégorie et des exemples
  5. Cliquez sur "💾 Ajouter le mot"

✏️ Pour modifier des mots :

  1. Allez dans "📝 Gérer les mots"
  2. Trouvez le mot à modifier
  3. Cliquez sur le bouton "✏️" à côté du mot
  4. Modifiez les informations
  5. Sauvegardez vos changements

💾 Pour sauvegarder :

  1. Allez dans "📁 Import/Export"
  2. Cliquez sur "💾 Télécharger JSON"
  3. Gardez ce fichier en sécurité !
💡 Astuce : Vos modifications sont automatiquement sauvegardées dans le navigateur. Pensez à télécharger une sauvegarde régulièrement !
